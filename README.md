
# Simple Encrypt/Decrypt Website 🟢

It's a web and python based encryption website that you can encrypt/Decrypt anything 🔥


## Features

- Able to Encrypt and Decrypt Text 💬
- Add your own python files 🗃️
- Live Tool ⭕
- Cross platform 📲
